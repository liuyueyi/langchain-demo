## 🔧 技术栈概览

### 核心框架
- **LangChain**：主要AI应用开发框架
- **uv**：现代化Python包管理器
- **Pydantic**：数据验证与结构化处理

### 关键技术
- **环境管理**：dotenv配置分离
- **模型集成**：OpenAI风格API调用
- **交互模式**：同步/流式/批量调用
- **对话管理**：状态维护与上下文处理
- **工具扩展**：外部系统集成能力
- **数据结构**：标准化输出格式
